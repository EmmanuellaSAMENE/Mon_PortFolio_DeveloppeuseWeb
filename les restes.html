Si tu veux que **EMMACARS reste sur une seule ligne** ET que le logo soit à côté du texte, il faut **mettre le texte et le logo dans le même conteneur flex** et utiliser l’ordre que tu veux. --- ## ✅ Solution HTML ```html
    EmmaCars
    ``` * Ici, **logo à gauche**, texte à droite * Tu peux inverser l’ordre si tu veux texte à gauche et logo à droite --- ## ✅ CSS pour que tout soit sur la même ligne ```css .logo-container { display: flex; /* met tout sur la même ligne */ align-items: center; /* centre verticalement */ gap: 10px; /* espace entre texte et logo */ } .logo-img { width: 60px; height: 60px; border-radius: 50%; /* rond */ object-fit: cover; } .logo-text { font-weight: 900; font-size: 24px; white-space: nowrap; /* empêche le texte de se casser */ } .logo-text span { color: red; /* pour le style EmmaCars */ } ``` --- ### 🔹 Explications importantes 1. `display: flex` → texte et image **toujours sur la même ligne** 2. `white-space: nowrap` → empêche que “Emma” et “Cars” se cassent sur deux lignes 3. `gap` → espace entre le logo et le texte --- Si tu veux, je peux te faire **la version finale de ton header complet** avec : * Logo rond + texte sur la même ligne * Menu avec liens “Accueil, Véhicules…” * Bouton LOGIN * Responsive mobile 🔥 Tout prêt à copier-coller.
